# Day 5 Exercise – Scientific Method

## Objective

Design a simple experiment; from your **research questions**, which you have designed as _stories_, extract viable **hypotheses** to test them. Using these hypotheses, design an experiment; then select the method to perform the expriment. All of which should be _documented_, ensuring _reproducability_, in the IMRaD report.

This is part of the _group work_.
